Abandoned User Stories

  • As a user, I want to be able to select different airlines.
  • As a user, I want to be able to choose a type of room, to meet my requirements.
  • As a user, I want to be able to read a short description of the hotels, so I can learn a little bit about them without having to visit their homepages.
  • As a webmaster, I want to be able to view all flights that have been booked on the website so I can assess profits.
  • As a user I would like to be able to filter search results by flight types e.g. business class or first class. So I can separate myself from the peasants.
